Política de Privacidad

Última actualización: 20 de marzo de 2026

1. Información que recopilamos

Curiosity ha sido diseñada con la privacidad como prioridad absoluta. La aplicación **no recopila, almacena ni transmite ningún tipo de información personal identificable** de sus usuarios.

2. Uso de APIs de terceros

La aplicación obtiene datos curiosos de servicios externos (como UselessFacts API, Numbers API y API Ninjas). Estas peticiones se realizan de forma anónima y no incluyen datos del usuario.

3. Traducción de contenidos

Para la traducción de los datos, utilizamos el Framework nativo de Apple (iOS 18+) o servicios de terceros como DeepL y MyMemory. Estas traducciones se procesan sin vincular ninguna identidad personal.

4. Almacenamiento Local

Curiosity almacena tus preferencias de tema y un historial de datos vistos **únicamente en tu dispositivo**. No utilizamos servidores propios para almacenar esta información.

5. Cambios en esta política

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Te recomendamos revisar esta página periódicamente para cualquier cambio.

6. Contacto

Si tienes alguna pregunta sobre esta política de privacidad, puedes contactarnos en: jcaboroca@gmail.com
